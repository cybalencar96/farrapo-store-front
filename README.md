### **Prerequisites**

- npm

<br />

### **Installation**

1. Clone repository

```sh
git clone https://github.com/cybalencar96/farrapo-store-front.git
```

2. Install dependencies executing command in root

```sh
npm i
```

<br />

### **How to run**

```sh
npm start
```
